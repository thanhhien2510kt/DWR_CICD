# UI Debug Agent

## Description

This skill enables the agent to inspect web applications using Chrome debug tools.

The agent can analyze DOM elements, identify stable locators, and assist in generating UI automation scripts.

---

## When to Use

Use this skill when tasks involve:

- exploring UI elements
- debugging UI automation tests
- analyzing DOM structure
- identifying stable element locators

---

## Inputs

The skill may receive:

Application URL  
DOM elements  
HTML snippets  
UI steps  

---

## Output

Locator suggestions  
UI element analysis  
Automation recommendations
